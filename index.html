我的JS学习笔记12月22日

我的JS学习笔记12月22日

1.函数相关

  • 多个脚本或函数在调用时一定要通过依赖关系注意引入的顺序。
  • 作用域：即函数或变量的有效范围。限定了那部分代码可以访问该变量或方法。
  • 一般来说稍微避免全局变量和函数的创建，因为其很容易被改变，可以通过一些方法来减少这种风险，比如闭包。
  • 函数内变量的生存期是仅在函数运行期间，运行结束后就将被JS的垃圾回收机制回收。全局变量的生存期就是页面的生存期。
  • 标识符：即变量的名称，JS的查找机制一般是先从作用域内查找，如无，即从下一个更大的作用域查找，以此内推。因而常在函数内申明变量，即为了减小JS查找标识符的内存消耗。

    2.关于对象

  • 对象有对应的属性和方法。例如车，颜色是属性，加速是方法等。JS中如Data对象，getTime()即是他的方法，能够计算调用时到1970年1月1日0点的毫秒数。
  • 创建对象。例如 var mydate = new Date();其中new告诉JS创建一个新对象，Date()即使Date对象的构造函数。
  • 注意数据存储时(特指对象等，和基本的文本，数字不同，文本数字是直接存储的数据)，并不是在变量中存储了数据，而是在变量中存储了这个数据（对象）的内存地址。例如当一个对象有两引用时，通过一个改变对象里的值，另一个也会改变，因为他们同时引用相同的地址。

内置对象常用方法

字符串

  • indexOf(),从前面开始找，lastIndexOf(),从后往前开始找，也可以理解为最后出现的位置,且两个方法第二个参数都可以输入开始查找的位置（也可以用在数组上）：

    "abcda".indexOf(a) //0

    "abcda".lastIndexOf(a) //4

  • substring(),两个参数，第一个为起始位置，第二个结束位置，substr()，第一个起始位置，第二个是截取长度，两个方法都是返回一个字符串。

  • toLowerCase()/toUpperCase()，转换大小写。
  • charAt(),接受一个参数，返回这个索引位置的字符，charCodeAt(),是返回特定位置的Unicode码，比如A的编码是65，fromCharCode(65)返回A，跟charCodeAt()正好相反。
  • trim(),去掉字符串两端的空格。

数组

  • concat(),连接两个单独的数组，返回一个新数组。
  • slice()，返回一段数组，接受两个参数，为开始索引和结束索引，返回的数组包含前者不包含后者。
  • join(),将所有元素做为字符串返回，接受的参数为中间可以间隔的元素。
  • sort(),将数组按照元素的unicode码排序，大写字母的序列在小写字母序列的前面。
  • reverse(),数组反序。
  • 循环函数：

测试型：every();some();filter();操作型：forEach();map().

var number = [1,2,3,4,5];
function isLessThan3(value,index,array)
{
var resultValue = false;
if(value < 3) returnValue = true;
return returnValue;
} 
alert(number.every(isLessThan3));

值得注意的是isLessThan3这个函数接受的三个参数，都是必须的，此为固定格式，在所有的数组循环函数中都是相同的。
every()要求数组中每个函数都要满足限定的条件，最终才会返回true，显然这里返回的是false;换成some(),就会返回true，因为它只要求有元素满足就可以。filter()则是取出这些元素，最后以一个数组形式返回。

function doubleNumber(value,index,array)
{
    var result = value * 2;
}
number.forEach(doubleNumber);

这样就将数组中的每个元素加倍了。但是forEach()没有返回值，map()跟forEach()的作用相同，但是在操作数组的同时会复制一份结果数组来作为返回值。

Math对象

Math对象不需要创建，JS会自动创建。

Math.PI = π；

Math.abs() 求得绝对值；

Math.min(1,2) // 1; 取最小值
Math.max(1,2) //2; 取最大值，两个函数都接受多个参数

Math.ceil(); 向上取整，注意正负，(-1.1) // -1;
Math.floor();向下取整
Math.round(); 四舍五入
Math.parseInt(); 取整

Math.random(); [0-1)取随机数

Math.pow(2,3); 2的3次方。可以用来取N位小数。例：

var n = Math.pow(10,n);
fixNumber = Math.round(fixNumber * n) / n;

Number对象的toFixed方法可以直接实现这个功能，5.245.toFixed(2) //5.25

Date对象

var theDate = new Date() ; 未传入参数的时候就以运行此脚本的PC的本地时间为准。

也可以直接传入时间 :

new Date(“31 January 2014”);

new Date(2014,0,31,15,35,20,20); 年月日时分秒毫秒，注意1月是0,12月是11。

保存Date对象后，可以通过一些方法取得时间

getDate() 月份中的第几天

getDay() 星期几，日是0，一是1

getMonth() 月份，一月是0

getFullYear() 返回四位数年份

toDateString() 基于当前市区，返回一个便于阅读的时间，如”Wed 31 Dec 2003”

设置日期：

setDate();月中某一天 setDate(32);会滚动到下一月
比如要知道28天后，可以

var a  = Date.getDate();
Date.setDate(a + 28); //也可以减28,获得28天前的日期

setMouth();某一月

setFullYear();某一年

一天内的日期：

getHours();setHours(); 同样可以自动滚动

getMinutes();setMinutes();

getSeconds();setSeconds();

getMilliseconds();setMilliseconds();

toTimeString(); // “13:03:51 UTC”

自定义对象

function Person (firstName,lastName)
{
    this.firstName = firstName;
    this.lastName = lastName;
}

上面的代码建立了一个原型（prototype）,注意并没有实例出对象。

new Person("Qingye","Wang"); //实例出对象

每个对象都有prototype属性，可以通过这个属性来编辑原型：

Person.prototype.getName = function(){ return this.firstName }

in和hasOwnProperty的区别是，in是只要实例上有就会返回true,而后者则必须是存在于实例上的，继承的不算。通过delete可以删除实例上的属性，还原回原型属性。
